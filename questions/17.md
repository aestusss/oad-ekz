# Направления визуализации данных. Сравнение. Сравние части с целом. Временное сравнение
Визуализация даннызх - "мост" между сырыми данными и управленчискими решениями. Основные напрвавления класифицируются по задачам, которые они решают:

## 1. Сравнение
Используется, когда нужно выявить различия между объектами или категориями

*Цель* - сопоставить значения между несколькими категориями или объектами в один момент времени
*Примеры используемых графиков:*
*   Столбчатая диаграмма - вертикальная (для небольшого числа категорий) или горизонтальная (если названия категорий длинные)
*   Группированная диаграмма - для сравнения нескольких показателей внутри одной категории
*   Лепестковая диаграмма - для сравнения объектов по нескольким параметрам одновременно

## 2. Сравнение части с целым

Показывает, какую долю занимают отдельные элементы в общем объеме данных
*Цель* - проанализировать структуру данных и выявить наиболее значимые компоненты
*Примеры используемых графиков:*
*   Круговая диаграмма - эффективна при малом количестве долей и при большой разнице между ними (часто вместо нее используется кольцевая)
*   Древовидная карта - подходит для отображения иерархии структуры из большого числа элементов
*   Нормированная гистограмма - походит для сравнения структур нескольких групп друг с другом

## 3. Временное сравнение
Используется для отображения динамики изменения показателей за определенный период
*Цель* - выявить трендлы, сезонные колебания, циклы, аномальные скачки
*Примеры используемых графиков*: 
*   Линейный график - стандарт для отображения непрерывных данных
*   Диаграмма с областями - акцентирует внимание на общем объеме накопленного изменения под линией тренда
*   Ступенчатый график - используется, когда изменения происходят дискретно
*Особенность* - время всегда распологается по оси X, слева направа

# Данные, информация, знания. Пирамида. Отличие информации от данных
В информатике и менеджменте знаний иерархическая связь между этими понятиями описывается моделью
DIKW (Data, Information, Knowledge, Wisdom)

## 1. Пирамида знаний DIKW
Представляет из себя пирамиду, где каждый уровень является фундаментом для следуюшего

1) *Данные* - основание пирамиды. Разрозненные факты, цифры, симыолы сигналы. Лишены контекста и смысла
2) *Информация* - структурированные данные, прошедшие обработку и получившие контект
3) *Знания* - результат анализа информации и ее интеграции с опытом, пониманием того, как ее использовать
4) *Мудрость* - высший уровень. Понимаение причинно-следственныз связей и этических аспектов для принятия решений

| Характеристика | Данные | Информация | Знания |
| :--- | :--- | :--- | :--- |
| Что это? | Разрозненные факты | Обработанные данные | Совокупность опыта и информации |
| На какой вопрос отвечает? | Что? | Кто? Где? Когда? | Как использовать? Почему? |
| Пример | Число 1984 | "Цена продукта - 1984$" | "При цене 1984$ - продукт переоценен" |
| Уровень в IT | БД, файлы | Отчеты, дашборды | Экспертные модели |

## 2. Отличие информаци от данных
Ключевое различие заключается в наличии смысла и полезности

| Характеристика | Данные | Информация | 
| :--- | :--- | :--- | 
| Природа | Сырье, набор фактов | Обработанный продукт | 
| Контекст | Отсутствует | Наличие контекста и связей | 
| Структура | Часто неорганизованы | Организованы и структурированы |
| Полезность | Минимальна без обработки | Высокая - основа для приняьия решений | 
| Пример | Список чисел 1, 3, 3, 7 | "Среднее число заказов в час равно ~3.5" |
